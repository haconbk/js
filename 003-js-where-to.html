JavaScript Where To

JavaScript được chứa trong
  • Trong cặp thẻ script
  • Trong thẻ head
  • Trong thẻ body
  • ..và chứa ở file bên ngoài

Một số chú ý

Placing scripts at the bottom of the <body> element improves the display speed, because script interpretation slows down the display.

External scripts cannot contain <body> tags.
